## Two tracks, one site

| Track | Who | What they need |
| :--- | :--- | :--- |
| **Professional** | Scholars & educators | Searchable catalogue, citable images, exhibition history, self-service. |
| | Curators & museum staff | Where objects are, rights info, exhibition planning. |
| | Grant applicants & recipients | Clear guidelines, simple application, track record. |
| | Peer foundations | Evidence WFAA is serious, partnership routes. |
| **Public** | Enthusiasts, novice to expert | All of Andrew Wyeth's work in one place, a way in. |

<p class="opacity-85 mt-2">Both matter, and they reinforce each other.</p>

<Quote cite="Robin West">Novice or expert. For one we need to make a first impression; for the other, we need to change perceptions.</Quote>
